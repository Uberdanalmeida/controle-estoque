# Sistema de Estoque

Projeto desenvolvido para gerenciamento de produtos com dashboard interativo.

## 🚀 Funcionalidades

- **Registo de Produtos:** Adição de novos itens com validação de campos.
- **Gestão de Stock:** Listagem, edição e remoção de produtos diretamente na interface.
- **Pesquisa Inteligente:** Filtro dinâmico por nome, descrição ou categoria.
- **Feedback Visual:** Alerta de stock baixo (menos de 3 unidades) e notificações "Toast" de sucesso.
- **Persistência de Dados:** Integração total com base de dados cloud Supabase.
- **Design Responsivo:** Interface limpa e organizada para uso profissional.

---

## 🛠️ Tecnologias Utilizadas

- [React.js](https://reactjs.org/) - Biblioteca para a interface.
- [Supabase](https://supabase.com/) - Base de dados PostgreSQL e Autenticação.
- [Vite](https://vitejs.dev/) - Tool de build rápida para frontend.
- [React Icons](https://react-icons.github.io/react-icons/) - Conjunto de ícones personalizados.
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização personalizada.
