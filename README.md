# Sistema de Estoque

Projeto desenvolvido para gerenciamento de produtos com dashboard interativo.

## Funcionalidades
- Cadastro, edição e remoção de produtos
- Dashboard com totais automáticos
- Busca em tempo real
- Persistência em LocalStorage
- Modal com animações e UX profissional

## Tecnologias
- React
- CSS moderno
- Hooks
- LocalStorage API
